About the Project

The Smart Medicine Reminder System is designed to help individuals manage their medication schedules effectively. It utilizes an Arduino microcontroller to trigger alarms at user-defined times, ensuring that medications are taken consistently and on time.

How It Works

The system consists of a microcontroller connected to a user interface, which allows users to set alarms and receive notifications. Hereâ€™s how it works:

  1. User Interface: Users can set alarm times through a simple web interface.
  2. Alarm Triggering: At the designated times, the system triggers an alarm to remind the user to take their medication.
  3. SMS Notifications: If a user misses a dose, the system can send an SMS notification to a caretaker or family member, ensuring accountability.
  4. Daily Health Challenges: The system encourages users to participate in daily health challenges to promote better health habits.
  5. Health Facts: Users receive random health facts each time they visit the site, providing them with valuable information about wellness.

Benefits

  • Improves medication adherence, reducing the risk of missed doses.
  • Encourages healthier lifestyle choices through daily challenges.
  • Provides users with important health information to support their wellness journey.
  • Enhances communication between patients and caregivers.
